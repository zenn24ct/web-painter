- # ✨ Advanced Web Painter ✨

シンプルなブラウザベースのペイントアプリケーションです。複数のレイヤーを使い分け、描画、消去、および作品のPNG形式での保存が可能です。

#### 主な機能 (Features)

* **マルチレイヤー描画**: 複数の透明なレイヤーを重ねて描画し、複雑な編集を可能にします。
* **描画ツール**: 自由な線描画が可能です。
* **消しゴムツール**: アクティブなレイヤーの描画内容を部分的に消去できます。
* **ブラシ設定**: 描画色と線の太さ（ブラシサイズ）を調整可能です。
* **レイヤー操作**: レイヤーの追加、削除、アクティブレイヤーの切り替えが可能です。
* **画像保存**: 完成した作品（結合された全レイヤー）をPNGファイルとしてダウンロードできます。
* **クロスデバイス対応**: マウス操作だけでなく、タッチ操作（モバイルデバイスなど）にも対応しています。

#### 使い方 (Usage)

1.  **ファイルの配置**: `index.html`, `style.css`, `script.js` を同じディレクトリに配置します。
2.  **アプリ起動**: `index.html` をWebブラウザで開きます。
3.  **描画**: パレットから色と太さを選択し、キャンバス上でドラッグして描画します。
4.  **レイヤー切り替え**: 右側のレイヤーパネルで、描画したいレイヤー名をクリックしてアクティブ（選択状態）にします。
5.  **消去**: 「🧽 消しゴム」ボタンをクリックし、キャンバスをドラッグすると描画内容が透明になります。
6.  **保存**: 「💾 画像保存 (PNG)」ボタンをクリックすると、結合された画像がダウンロードされます。

#### 技術スタック (Tech Stack)

#### 今後の改善点 (Potential Enhancements)

現在のアプリケーションをさらに高機能で使いやすくするために、以下の改善が考えられます。

##### 1. ユーザビリティ・UI/UXの改善

* **Undo/Redo機能**: 操作履歴を保存し、直前の操作を取り消し（元に戻す）またはやり直し（進める）できるようにする。
* **レイヤーの並び替え**: レイヤーパネル内でドラッグ＆ドロップでレイヤーの重なり順を変更できるようにする。
* **レイヤーの透明度/可視性**: 各レイヤーごとに透明度（Opacity）を設定したり、一時的に非表示（トグル）にしたりする機能。
* **直感的なUI**: ツールのアイコン化、スライダー操作時のリアルタイムプレビュー（例: ブラシサイズのプレビュー）。
* **モバイルUIの最適化**: ツールバーやレイヤーパネルを画面サイズに応じて適切に配置する（特に縦長画面での操作性の向上）。

##### 2. 描画ツールの拡充

* **図形描画**: 直線、長方形、円などのプリミティブな図形を描画できるツール。
* **塗りつぶしツール**: クリックした領域を単一の色で塗りつぶす（バケツ）ツール。
* **スポイトツール**: キャンバス上の特定の色を拾って、現在の描画色に設定する機能。
* **ブラシオプション**: エアブラシ、カリグラフィなど、線の形状やテクスチャを変更できるオプション。
* **描画のスムージング**: 線のブレを軽減し、より滑らかな線を描画する処理（スタビライザー）。

##### 3. パフォーマンスと機能の強化

* **ズーム/パン機能**: キャンバスを拡大・縮小したり、移動したりして、細かい部分の描画を可能にする。
* **Web Workers**: パフォーマンスが要求される処理（例: 複雑なフィルタ処理、画像合成）をメインスレッドから分離し、UIの応答性を保つ。
* **画像形式の選択**: PNGだけでなく、JPEGやWebPなどの他の画像形式での保存オプションを追加する。
* **ローカルストレージへの自動保存**: 描画データをブラウザのローカルストレージに定期的に保存し、ブラウザを閉じても作業を継続できるようにする。
* **ファイル読み込み**: 既存の画像ファイル（PNG, JPEGなど）を新しいレイヤーとして読み込む機能。

##### 4. 高度なレイヤー操作

* **レイヤー名の変更**: ユーザーがレイヤー名を自由に編集できるようにする。
* **レイヤーの複製**: 既存のレイヤーを簡単に複製して、編集のバリエーションを作成できるようにする。
* **レイヤーの結合**: 選択した複数のレイヤーを1つに統合する機能。
